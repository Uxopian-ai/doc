# Admin Dashboard

The **Uxopian-ai Admin Panel** acts as the central command center for your AI infrastructure. It provides a visual interface to manage resources, monitor usage statistics, and oversee user activity.

!!! info "Access"
    The Admin Panel is restricted to users with the **admin role**.
    **URL:** `https://<your-uxopian-endpoint>/admin`

## Key Features

The dashboard offers quick access to the primary management modules:

- **[Prompts](prompts.md):** Manage, test, and analyze AI prompt configurations.
- **[LLM Providers](llm_providers.md):** Configure and test LLM provider connections.
- **[Statistics](statistics.md):** View global health metrics, usage trends, and ROI data.
- **[Users](users.md):** Monitor user activity and view detailed interaction history.
